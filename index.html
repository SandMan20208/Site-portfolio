Проекты

Учебный проект написан на PHP-фреймворке Yii2. Состоит из двух модулей: пользовательского и административного. Пользовательская часть представляет собой сайт провайдера цифровых услуг с возможностью оставлять заявки на подключение услуг через форму обратной связи. Административная часть приложения предоставляет функционал, позволяющий управлять учётными записями работников, наполнением рекламных блоков сайта и новостной ленты, регистрировать, редактировать и удалять входящие заявки от клиентов. Административная часть приложения не имеет адаптивной верстки, создавалась для использования на ПК.

Приложение написано на чистом PHP с использованием базы данных MySQL, представляет собой интерфейс для работы с базой данных пропусков, позволяет добавлять, редактировать, удалять пропуска, а так же отслеживает пропуска, срок действия которых заканчивается.

Интерфейс простой, разрабатывался для неопытных пользователей ПК. Приложение расчитано для работы только на ПК, не имеет адаптивной верстки.
